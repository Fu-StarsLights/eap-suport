EdgerOS 特权态 EAP 开发工具

功能：
    1. 更新特权态 eap 应用
    2. 接收 EdgerOS 应用日志

安装:
    npm install -g git+https://github.com/Fu-StarsLights/eap-support.git

执行:esport 指令

> 注意
运行 esport 上传文件时,需要在包含特权态 eap 完整的结构目录中。
